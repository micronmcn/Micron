What is MICRON

MICRON is a user-friendly website concept built for smooth registration, login, and personal dashboard access. It is designed with a simple visual style so users can understand everything quickly. The structure keeps important actions visible and easy to use. It also supports popups for sign up, policy details, and menu links. The goal is to create a clean and modern experience.
